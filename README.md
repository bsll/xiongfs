-git checkout -b newbranch 新建分支
-git checkout abranch  切换到某个分支
-git branch 查看分支(加 * 表示当前所在分支)
-git branch -d tmpbranch  删除临时分支
-git merge tmpbranch  将tmpbranch分支合并到当前所在的分支
